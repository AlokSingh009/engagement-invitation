# Engagement Invitation — Alok & Shephali

A scroll-driven engagement invitation styled like the reference video — floral envelope, temple pillars, save-the-date countdown, and venue details.

## How to view

Open `index.html` in any modern browser.

## Features (inspired by reference video)

1. **Maroon floral envelope** with wax seal — opens on scroll
2. **Golden temple pillars** with bells and Sanskrit blessing
3. **Couple names** in cursive with parent details
4. **Save the Date** with live countdown to 11 July 2026
5. **Welcome card** with glassmorphism style
6. **Venue details** with Google Maps directions button
7. **Falling rose petals** animation
8. **Background music**
